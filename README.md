## 💡 서비스 소개

### 차량 도감 수집 어플리케이션

> 차종을 모르겠어 ! ==> 사진을 찍으면 차량의 정보를 알려주자
>
> 내가 본 자동차를 기록하고 싶어! ==> 자동차를 도감 형식으로 수집하게 하자
>
> 교통 법규나 표지판이 헷갈려 ==> 자연스럽게 지식을 습득하게 하자
